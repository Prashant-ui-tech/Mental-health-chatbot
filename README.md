# Mental-health-chatbot
Our project is an AI-based Mental Health Chatbot that helps users express their feelings and get supportive responses. It is built using Python, FastAPI, and OpenAI API. The chatbot can understand user messages and reply accordingly. It will also include crisis detection to provide safer and helpful responses to users.

This project is an AI based mental health chatbot that helps users talk about their feelings and receive supportive responses. The chatbot will be built using Python, FastAPI and OpenAI API.

Features planned:
- Chatbot conversation support
- Crisis message detection
- Session memory
- Document based Q&A using Llama Index
